# scoop-bucket

Scoop bucket for monkescience projects.

## Usage

```
scoop bucket add monkescience https://github.com/monkescience/scoop-bucket
scoop install <app>
```

## Available Apps

| App | Description |
|-----|-------------|
| [yeet](https://github.com/monkescience/yeet) | Automate releases based on conventional commits |
